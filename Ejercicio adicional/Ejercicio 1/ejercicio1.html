Resumen de caracteristicas disponibles en HTML 5

  1. Semántica mejorada: Introdujo nuevas etiquetas semánticas como <header>, <footer>, <nav>, <article>, <section>, entre otras, para facilitar la estructura y comprensión del contenido.

  2. Elementos de audio y video: Incorporó las etiquetas <audio> y <video> para integrar contenido multimedia en las páginas web sin necesidad de complementos

  3. Canvas y SVG: Introdujo <canvas> para gráficos dinámicos y dibujos, y permitió la integración de gráficos vectoriales escalables (SVG) directamente en las páginas web.

  4. Formularios mejorados: Presentó nuevos tipos de entrada para formularios, como email, url, date, number, entre otros, junto con validaciones y controles más avanzados.

  5. Almacenamiento local: Introdujo el almacenamiento local (localStorage y sessionStorage) para permitir que las aplicaciones web almacenen datos en el navegador del usuario.

  6. Geolocalización: Proporcionó soporte para obtener la ubicación del usuario a través del API de geolocalización.
  7. Web Workers: Permitió la ejecución de scripts en segundo plano mediante hilos separados, lo que ayuda a mejorar la eficiencia y capacidad de respuesta de las aplicaciones web.

  8. APIs de accesibilidad: Introdujo nuevas características y mejoras en la accesibilidad web, como el atributo alt mejorado para imágenes y elementos asociados con la accesibilidad.

  9. APIs de drag-and-drop: Incorporó capacidades nativas para arrastrar y soltar elementos en la página web.

  10. Soporte offline: Permitió el desarrollo de aplicaciones web que funcionan sin conexión a internet, mediante el uso de la API de caché de aplicación (AppCache).
