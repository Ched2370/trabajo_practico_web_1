Investigacion

HTML

HTML6: Aunque no había una fecha clara para su lanzamiento, se esperaba que HTML6 continuara expandiendo la semántica y la funcionalidad del lenguaje, posiblemente con más enfoque en la accesibilidad, formularios avanzados y API más potentes.


CSS

CSS Grid y Flexbox:
Se esperaba una mayor adopción y madurez de CSS Grid y Flexbox para la creación de diseños web responsivos y complejos.

CSS Variables:
Su uso se estaba extendiendo, permitiendo una gestión más dinámica y eficiente de estilos.

Nuevas características de diseño:
Se exploraban opciones como subgrids (para anidar grids), aspectos de diseño más avanzados y capacidades de estilo más sofisticadas.

JavaScript (ECMAScript)

ECMAScript 2022 y más allá:
Se preveía la continua evolución de JavaScript con nuevas funcionalidades, mejoras de rendimiento y soluciones para desafíos específicos de desarrollo.

Mejoras de rendimiento y WebAssembly:
Se estaba trabajando para mejorar el rendimiento de JavaScript y su integración con WebAssembly para ejecutar código de alto rendimiento en el navegador.

TC39 Proposals:
Nuevas propuestas de TC39 para añadir funcionalidades al lenguaje, como el pipeline operator, registros privados, manejo de errores más avanzado, entre otros.
